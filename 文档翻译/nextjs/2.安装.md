# Installation

System Requirements:

- [Node.js 18.17](https://nodejs.org/) or later.
- macOS, Windows (including WSL), and Linux are supported. (WSL（Windows Subsystem for Linux）是一种在Windows操作系统上运行Linux环境的兼容层)
